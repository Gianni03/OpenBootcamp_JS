# OpenBootcamp JS

🇪🇸 **Repositorio del Curso de JavaScript en OpenBootcamp** 📚

Este es el repositorio del curso de JavaScript en OpenBootcamp, que contiene las clases y ejercicios realizados. Durante el curso, se cubrieron los siguientes temas:

- Sets
- Promesas
- Objetos
- Métodos
- Clases
- Arrow Functions
- Manejo de Arrays
- Módulos
- Eventos
- ESlint

En este repositorio encontrarás el material correspondiente a cada tema, incluyendo ejercicios prácticos y ejemplos de código. Si estás interesado en aprender JavaScript y desarrollar tus habilidades en estos conceptos, te invito a explorar el repositorio y aprovechar el contenido disponible.
